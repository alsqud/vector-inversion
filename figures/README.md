# Figures

This directory stores generated figures for the paper and reproducibility checks.

## Generated Figures

The following figure may be generated by the plotting script:

```text
figures/tradeoff_hit1_cosine.png
```

## File Descriptions

| File | Description |
|---|---|
| `tradeoff_hit1_cosine.png` | Trade-off plot between retrieval performance and inversion risk |

## Generation Script

To generate the trade-off figure, run:

```bash
python scripts/05_plot_tradeoff.py --config config/default.yaml
```

By default, the script uses:

```text
results/seed_repeat_summary_merged.csv
```

and saves the figure to:

```text
figures/tradeoff_hit1_cosine.png
```

## Git Tracking Policy

Generated figure files may be tracked when they are used for paper reproduction.

However, intermediate visualization outputs or temporary plots should not be committed unless they are directly related to the paper results.
